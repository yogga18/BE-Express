HTTP Status Code https://http.cat/

Successfull Responses (2xx)
* 200 OK (GET, PATCH)
* 201 Created (POST, PUT)

Client Error Response (4xx)
* 400 Bad Request
* 401 Unauthorized (Auth Case)
* 403 Forbidden
* 404 Not Found
* 405 Method Not Allowed

Server Error Response (5xx)
* 500 Internal Server Error
